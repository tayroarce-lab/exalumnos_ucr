feat(cv): Implementación completa del Módulo RF-11 (Curriculum Vitae ATS)

Este commit integra todo el sistema de creación, edición y exportación de Currículum para estudiantes:

1. Base de Datos (Supabase):
- Migración SQL con tablas modulares (perfiles, academia, experiencia, habilidades, certificaciones).
- Seguridad RLS habilitada para garantizar la privacidad de la información.
- Triggers a nivel de BD para asegurar que las viñetas de experiencia tengan menos de 120 caracteres.

2. Backend (Server Actions & Rutas):
- Server actions tipados con Zod para el autoguardado de cada formulario (creación y borrado atómico).
- Ruta de Inteligencia Artificial conectada a gpt-4o-mini para mejorar la redacción de viñetas al estándar ATS.
- Ruta de PDF con '@react-pdf/renderer' para generar y exportar el currículum en tiempo real.

3. Frontend (UI & UX):
- Página principal en '/dashboard/cv' con redirección de seguridad inicial.
- Editor interactivo (CVStepper) dividido en 4 pasos amigables.
- Formularios reactivos con autoguardado en segundo plano sin interrumpir al usuario.
- Componentes especializados: TagInput para habilidades y BulletInput integrado con IA.
